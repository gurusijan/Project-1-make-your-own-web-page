Coding languages for building a website

Building a fancy website happens in several levels. For the strucure and the content of the website you use HTML. HTML works with tags which can also include other tags between the opening and closing tags. The style of the elements (opening tag + content + closing tag) is done in the language CSS (comparable with the decoration in a house). For interactive elements you use the language java script. The coding elements of a website can be seen with the developer tools of Google Chrome.

How to build a website

The coding of websites is organized in tree-like-structures. Each element is in one line. This structure is called Document Object Model (DOM). For the handling of the different elements on the website you use boxes. In the HTML code these are signed with the tag DIV. Using boxes makes it much easier to reorganize the different elements on your website and to remain an overview. Boxes are always rectangular even if they look like a circle or something else. The layout is done in CSS. Regularly you use boxes in boxes in boxes (from big to small). Another method to remain an overview of your coding is to label your boxes so that you still know what is in it. This is especially relevant if you use images and other objects with cryptic names like "pic1". Labels are done by using "class=" within the DIV tag.

How to use CSS (Cascading Style Sheets)

A cascade structure is a break down structure from big to small.It actually works like a waterfall (e.g. national law – state law – city law – house rules). A big advantage of CSS is the possibility of inheritance. Same structure-parts can be grouped with the same name by using the parameter “class” and for these you can set the same style rules. This functionality avoids a lot of unnecessary repeated coding which means less work and less complexity. In general descendant elements inherit text-related properties like colour, font, letter spacing and so on. But there are also properties which cannot be inherited like background, border and display.

How to include CSS style in a web page

How to include CSS style in my web page. There are actually three methods:
1. Write CSS in the head of the html file. This is good for small projects.
2. Link the HTML file to a separate CSS file. This is good for larger projects due to the better organization.
3. Write the style inline with your HTML. This is not suggested because of much unnecessary code and the high effort of keeping it up to date.

The Box Model

Boxes consist of several parts:
1. Content
2. Padding: Clears the area around the content
3. Border: is inherited from the colour of the box
4. Margin: is the transparent space between the boxes

Box Sizing

The box size is equal to the sum of the boarder, the padding and the content. The size can be written in pixels (px) or percentage. Additional you can use max width to ensure that it the box won’t destroy your layout. Another possibility is that instead of adjusting each single size of the box parts after any small change you can use the CSS code “box-sizing: border-box”. However you have to consider two things. First, margin is not included and second, it is a relatively new code and it has to be adjusted for different browsers. How to do that you can see in this video.

Why the speed of light is important for computers

The light travels about 299,792,458 metres per second. In one nanosecond light travels about 30cm. If a computer has a processor with 2.7 GHz this means that it can do 2.7 billion cycles per second. One cycle is one step. Calculated in light meters this means that the light travels 11.10…. meters during one cycle. This is also the reason why processors have to as small as possible.


Lecture 5

Variables in python

Instead of using big numbers or long characters you can define variables in python. The definition works through assignment statements which is done by using the equal sign. However in python the equal-sign is used in a different way than in maths. In python it means: put the value of the right side as meaning of the expression on the left side (e.g. days = days – 1 and days = 47; the value of days is 46 in this case). If you use a variable in a formula without assignment of a value an error will occur. Variables have to be defined. Usually variables are written in small letters. It’s not a must but a usual name convention.

Strings

A string is a sequence of characters surrounded by quotes. Strings are used to display plain text. This can be done with single (‘x’) or double quotes (“x”). In python strings and variable can be combined like:
-> Name = Janko
-> print “hello ” + name + “!”
After the “hello” must be a space otherwise name would be directly after it. Strings and numbers cannot be combined in a sum like “hello” + 9. This would lead to an error. Even if the string is number. If you take the sum of numbers in strings it looks like this -- “2” + “2” –> result is 22. However strings can be multiplied like -- print “!” * 12 -> result is !!!!!!!!!!!!. - Multi line strings can be created by triple single quotes like “”” or ‘’’. Then the text in the string can be written in several lines only with one print.

Using indexes

By adding an index behind a variable or a string only the specific characters would be displayed. Python counts the place of each character from zero on. For example by using the index [3] the fourth character of a variable or string would be displayed. By using negative numbers as in index python counts from the back. In this case [-1] would mean the last character. Using indexes you have to consider that python does the arithmetic part first (e.g. S[3] is equal to S[1+1+1].

Selecting sub-sequences from strings

There are different ways to use indexes. For example from – word = “assume”:
-> Print word [3] -> result: u
-> Print word [3:3] -> result: u
-> Print word [3:4] -> nothing bc there is no letter between the 3rd and 4th letter
-> Print word [4:6] -> result: me
-> Print word [4:] -> result: me (all letters after from no. 4 on
-> Print word [:4] -> result: assu (all letters until no. 4, not including 4)
-> Print word [:] -> result: assume (whole word)

Finding strings in strings

To find specific parts within a string you can use the procedure “find”. The output of the procedure is the number of the position. The usage of find looks like this – print “string”.find(“what to search for”). If nothing gets found then the output is -1. To define the position you want to search from you can add a second parameter to “find” like – “string”. Find(“string”, “8”) – means that python would search from postion 8 on. This procedure can also be used in variables to display specific parts. Here is an example of that: -> my_string = "My favorite color: blue"
-> color_start_location = my_string.find("color:")
-> favorite_color = my_string[color_start_location:]
-> print favorite_color[7:]
The result of this would be “blue”. More information about the procedure can be found in this video.

Functions (= procedure) in python

With a procedure you can map inputs to outputs. To define a function you have to use the expression "def". The whole function starts with a starting line including the "def", the name of the function and the inputs (Bsp.: def "name of function" (input1, input2)). After the starting line comes the body which includes the specification what to do with the parameters. At the end of the function you have to define what has to be returned. To use the function it is now just necessary to write the name of the function, followed by the defined value-parameters in brackets. Procedures can have x inputs which are also called operands or parameter or arguments. All inputs have to be used in a procedure. If a procedure has no return then it has no value and prints out "none".

Advantage of functions for programmers

With functions you can define a couple of programming steps which come over and over again. By using a function these steps can be done automatically with just one order like the definition of a DEV-Box in HTML. The individual content of the box would be defined by the parameters.

How to correct bugs

Bugs are part of every software development project. Hence the correction of them is part of the daily routine of every programmer. To correct bugs there are several possibilities you can use:
1. Examine error messages when program crash
a. The last line of python tracebacks provides information of what went wrong. If you read the message backwards then you can identify where to find the problem
2. Work from example code
a. If your individual code does not work then take that part out and try to use example code and which you can modify step by step until it does what you want.
3. Make sure examples work
a. Not all examples you find in the web are correct. You should always test example code first do ensure that it does what you want.
4. Check (print) intermediate results
a. If the code does not provide the expected outcome without crashing then add print statements to the program to see part results and to identify from which point on the program starts being incorrect.
5. Keep and compare old versions
a. Old versions of your programs should always be saved so that you always have a fall-back version. There are programs which help you with the administration of old version like Git-hub.

Lists

Strings are kind of structured data which can be broken down into their characters. However they are limited to characters. Instead of strings you can use lists which are much more powerful due to the wide range of possibilities. In lists you can use all kinds of elements like strings, numbers or lists inside a list. By using indexes you can just get one element of a list.

Mutation an Aliasing

Another big difference between strings and lists is that lists can be mutated. You can easily change the values of a list after you created it. If you replace one part of the list it still remains the same list just with a replaced element while if you redefine a variable with another string a total new variable is created. The mutation of a list can be done in two different way. First by using "append" (e.g. mylist.append(6)) and second by "+=" (e.g. mylist += [6]). If you do not want to mutate a list, just to enhance it for one procedure then you have to use the function form ""list" = "list" + [whatever]. One more big difference is that a list can be connected to two different variables. Changing a parameter by one of them would also change it for the other variable.

Loops in lists

With the expression "for "variable" in "list"" you can create a loop function for a list. The variable gets defined in the for-expression. The following body of the function contains the coding which gets assigned to every element of the list.

Index for lists

There are three possibilities to index the content of a list. First one is ""list".index("value")" which finds the position of a required value. If the value does not exist the output is an error. The second one is ""value" in "list"" which returns True or False. The third one is "value" not in "list" which also returns True or False.
