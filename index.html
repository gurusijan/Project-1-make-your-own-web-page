About the internet

The World Wide Web (www) consists of HTML (hyper text markup language) documents.
These documents are connected via hyper links with each other. It works like
this: The user sends a request by the browser on his computer via the internet
to the server and receives the documents. The request are done with HTTP
(hypertext transfer protocol). If you need more information about the internet
just have a look at this video.


HTML

Computers are stupid machines which need clear orders and information for
processing. HTML is the coding language which tells the computer what and
how it has to display the content and how it has to react on specific user
actions. HTML can be plain text, it can provide lay-outs (markup), it can
include other documents like pictures, videos, pdf, and it can contain links
to other websites.


HTML markup

The HTML markup consists of tags which are the orders tell the computer what
to do. The opening tag together with the content and (if necessary) the closing
tag are one element.


Tag types

There are different types of tags. Some tags need an opening and a closing
tag like “< b > content < / b >”. An anchor tag ads a hyper link to a defined word
or phrase like this one to the Udacity homepage. You can learn more about the
tag types by this fantastic video.
